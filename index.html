Cross-Device Augmented Reality Interactions for Mobile Phones and Head-Mounted Displays

This project presents a framework that transforms a mobile phone into an intuitive input interface for Mixed Reality. By transmitting real-time sensor data, touch inputs, and the device's 6-DoF pose to an MR headset, the system allows users to interact with virtual content more naturally.

Key Features:

  • Transforms mobile phones into intuitive input interfaces for Mixed Reality
  • Transmits real-time sensor data, touch inputs, and 6-DoF pose to MR headsets
  • Enables scenarios like pinching virtual objects from the phone's screen into the MR environment
  • Provides a versatile controller-free experience compared to traditional MR controllers

Exploring the Role of Tangible Objects in AR Tower Defense Gaming Interaction

This project explores the impact of tangible objects on user experience in Augmented Reality gaming, specifically within an self-designed AR tower defense game. While AR technology blends digital and physical worlds, most interactions rely on either freehand gestures or hardware-based inputs, each with its limitations.

Research Focus:

  • Exploring the impact of tangible objects on user experience in AR gaming
  • Designing a custom AR tower defense game as a research platform
  • Analyzing limitations of existing AR interaction methods (gestures and hardware inputs)
  • Proposing tangible interaction as a promising alternative to enhance immersion and provide more intuitive user experiences

Medieval Blade - An Action-Strategy Game

Medieval Blade is a group project developed as part of the ANU Game Development Course. Set in a medieval village, the game immerses players in a narrative where the village's various regions have been overrun by different types of monsters.

Game Features:

  • Players take on the role of heroes with different skills, tasked with battling monsters and reclaiming the village
  • Medieval village setting with different regions overrun by various types of monsters
  • Collaborative project developed as part of the ANU Game Development Course
  • Gameplay combining action and strategy elements
